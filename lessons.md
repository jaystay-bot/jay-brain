# lessons.md — Simple Rules

## Stack
- Next.js App Router only
- TypeScript only (.ts / .tsx)
- Tailwind + shadcn/ui + Lucide icons

## Backend
- Supabase with RLS
- Clerk for auth
- Stripe server-side only

## Rules
- Do not expose secrets client-side
- Do not disable RLS
- Do not use Pages Router

## Build Style
- One task at a time
- Minimal changes only
- Working > perfect
- Do not expand scope

## Testing
- Verify only what was changed
- Do not run full test suite unless asked

## Stop
- Stop when task is complete
- Do not continue into other fixes
